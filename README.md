# RC Engine

RC Engine é um simulador de redes em árvore desenvolvido para estudar diferentes estratégias de posicionamento de participantes em estruturas de marketing multinível.

## Objetivos

- Gerar árvores automaticamente.
- Calcular pais e filhos.
- Calcular níveis.
- Calcular descendentes.
- Simular múltiplas empresas.
- Comparar diferentes estratégias de cadastro.
- Gerar relatórios em Excel.

## Estrutura

- tests/
- data/

## Primeira versão

A versão 0.1 será responsável apenas pela geração da árvore e pelo cálculo dos descendentes.
